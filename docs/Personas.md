# Personas

## Primary Persona: Developer

### Responsibilities
- Build features
- Fix bugs
- Review code
- Write tests
- Maintain documentation

### Goals
- Understand work quickly
- Reduce context switching
- Deliver high-quality software
- Receive useful AI assistance

### Pain Points
- Scattered information
- Legacy code
- Missing documentation
- Poor visibility

### Questions
- What should I work on next?
- Where do I start?
- What changed?
- What APIs, docs and tests are affected?

---

## Primary Persona: Team Lead

### Responsibilities
- Plan work
- Assign tasks
- Review pull requests
- Monitor delivery
- Mentor developers

### Goals
- Improve team productivity
- Reduce blockers
- Maintain engineering quality

### Pain Points
- Lack of visibility
- Slow reviews
- Missing context
- Poor documentation
