# Non-Functional Requirements

## Performance
- Common API responses <300 ms.
- Search <1 second for common queries.

## Scalability
- Modular monolith initially.
- Designed for future horizontal scaling.

## Security
- JWT/OAuth2.
- Encrypted secrets.
- RBAC.

## Reliability
- Graceful failure.
- Automatic retries where appropriate.
- Audit logging.

## Maintainability
- Clean Architecture.
- SOLID.
- High test coverage.
- Documented APIs.

## Observability
- Structured logging.
- Metrics.
- Tracing.
- Health checks.
